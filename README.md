# Java Projects

This repository contains beginner Java console applications built for practice and learning core programming concepts.

## Projects Included

### 1. Expense Tracker
- Adds expenses
- Stores and calculates total spending
- Uses ArrayList and loops

### 2. To-Do List App
- Add tasks
- View tasks
- Remove tasks
- Uses ArrayList and menu system

### 3. Bank App
- Check balance
- Deposit money
- Withdraw money
- Uses conditionals and loops

## Technologies Used
- Java
- Scanner (user input)
- ArrayList
- Loops and conditionals

## Purpose
This project was built to practice Java fundamentals and prepare for internships in software development.
